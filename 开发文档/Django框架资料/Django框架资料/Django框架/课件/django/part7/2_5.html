搭建项目框架

  • 当前项目分支一共有3个，分别为master、dev、itcast，当前在itcast分支上工作
  • 在克隆的目录下创建项目，使用django框架

  • 将文件代码添加到暂存区
git add dailyfresh/
  • 将暂存区提交到仓储区
git commit -m '搭建框架'
